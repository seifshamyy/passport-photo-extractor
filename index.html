API Documentation

POST /api/extract

Upload a passport image file and receive the cropped photo.

Parameters:

  • passport - multipart/form-data file field
  • format - Query param: "base64" (default) or "binary"
  • fallback - Query param: "true" or "false" (default)
curl -X POST -F "passport=@passport.jpg" \ "https://your-app.railway.app/api/extract?format=base64&fallback=true"

POST /api/extract-base64

Send a base64 encoded image and receive the cropped photo.

Request Body (JSON):

{ "image": "data:image/jpeg;base64,/9j/4AAQ...", "format": "base64", "fallback": false }

Example:

curl -X POST https://your-app.railway.app/api/extract-base64 \ -H "Content-Type: application/json" \ -d '{"image": "data:image/jpeg;base64,...", "format": "base64"}'
